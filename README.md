# nayak-express-skeleton

This is a very lightweight Node.js Express REST API structure.

## Getting Started

### Installing

```
$ git clone git@github.com:LeeJeongYeop/nayak-express-skeleton.git
$ cd nayak-express-skeleton
$ npm install
```

### Start Server

```
$ npm start
```
